- Pull a branch in Git without checking it out: `git fetch <remote> <srcBranch>:<destBranch>` [[source](https://stackoverflow.com/a/5776415/1979008)]
- Get diff of last n _cached_ (staged) commits `HEAD` syntax: `git diff --cached HEAD~<n>`
- Use same filename as two arguments: `<cmd> <filename>{,}`
